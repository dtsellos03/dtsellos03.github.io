The Film Comparator

Node.js | Express | Socket.io

The Film Comparator analyzes a user’s IMDb ratings to generate insights into film preferences, such as favorite actors and genres, and presents these in a summary. Users submit their exported ratings file from IMDB and the application runs several sorting processes, as well as making API calls to the OMDb API to collect additional data.

My Dog's Mixtape

Node.js | Express | Angular 2 | MongoDB

MyDog’sMixtape is a single page application that catalogs dog breed mixes like the Pugle, a mix between a Pug and a Beagle. Users can search and filter by breed, and a voting system with corresponding top rated and lowest rated mixes is used. The mixes are stored in a MongoDB database, and users can submit their own mix.
